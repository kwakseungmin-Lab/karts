## 파이프라인 요약

```
1. 유튜브 다운로드 (forhonor_e3_2016.mp4)
2. 프레임 추출 (20장 스크린샷)
3. 영상 분석 → forhonor_analysis.md 작성
4. GPT Image 2 (gpt-image-2) → 8씬 스토리보드 이미지 생성
5. Sora sora-2 → 각 이미지를 레퍼런스로 영상 생성
6. FFmpeg → 최종 조립
```

## 생성 모델

| 단계 | 모델 |
|------|------|
| 스토리보드 이미지 | `gpt-image-2` (1536×1024 → 1920×1080) |
| 영상 생성 | `sora-2` (1280×720 → 1920×1080 업스케일) |
| 최종 편집 | FFmpeg concat |
